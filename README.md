## What is this?

Longevity Wiki is a minimalistic food encyclopedia that answers one question: **what should you eat to live a longer, healthier life?**

Every ingredient article is backed by specific biological mechanisms, cited studies, and consumption recommendations extracted from peer-reviewed longevity research. No opinions, no trends, no sponsored content. Just science.

- **59 ingredient articles** with health claims, mechanisms, preparation tips, and synergies
- **18 recipes** designed around scientific ingredient pairings (e.g., turmeric + black pepper for 2000% curcumin absorption)
- **101 ingredients** in the structured knowledge base, extracted from published research
- **236 health claims** with specific biological mechanisms and study references
- **126 ingredient synergies** mapped (what to combine and why)

## No advertising. Ever.

This wiki will never run ads, affiliate links, sponsored content, or paywalls. Nutrition information should be free, calm, and trustworthy.

## How it works

The content is generated by a pipeline of Claude Code skills that extract, research, and write:

```
Book (epub)
  → /extract-book-knowledge    → Structured JSON claims
    → /research-ingredient     → Enriched profiles with PubMed data
      → /generate-wiki-entry   → Readable wiki articles
      → /generate-recipe       → Science-backed recipes
```

The skills framework is open source too: [longevity-skills](https://github.com/manuelbenitez/longevity-skills)

## Tech stack

- **Next.js 16** with static export (zero server, pure HTML/CSS/JS)
- **Tailwind CSS** with custom bone-white design system
- **Fraunces** (serif display) + **Instrument Sans** (body)
- **Framer Motion** for entrance animations
- **Lenis** for smooth scrolling
- **JSON files** as the data layer (no database)
- **Markdown** wiki articles rendered server-side via remark

## Design

Minimalistic, bone-white (`#F5F0EB`), single sage-green accent (`#6B7F5E`). Hand-drawn SVG ingredient illustrations. No photography. The full design system is in [DESIGN.md](DESIGN.md).

## Development

```bash
git clone https://github.com/manuelbenitez/my-longevity-wiki.git
cd my-longevity-wiki
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Static export

```bash
bun run build
```

Generates static HTML in `out/`. Deploy anywhere: Vercel, Netlify, GitHub Pages, Cloudflare Pages, or any static file server.

## Contributing

Pull requests welcome. All changes require review before merging.

If you want to add a new ingredient or recipe, the easiest path is to use the [longevity-skills](https://github.com/manuelbenitez/longevity-skills) framework to generate the content, then submit a PR with the JSON profile and wiki article.
